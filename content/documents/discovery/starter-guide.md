# Discovery starter guide

Use this guide to establish shared context before proposing a solution.

## Suggested approach

1. Capture the desired outcome in observable terms.
2. Identify people, systems, policies, and constraints involved.
3. Separate known facts from assumptions.
4. Record unanswered questions and assign owners.
5. Link the resulting project output to the reusable entries that informed it.

## Agent guidance

An agent should ask for missing evidence, preserve uncertainty, and avoid turning assumptions into facts.
